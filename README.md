# Сенека — стоический компаньон

Личный сайт, чтобы **изучать, напоминать и применять** учения стоиков по
«Нравственным письмам к Луцилию» Сенеки.

Статичный сайт (Vite + React + TypeScript), без бэкенда. Все личные данные —
дневник, прогресс, избранное, стрик — хранятся в `localStorage` твоего браузера.

## Разделы

- **Сегодня** — мысль дня (детерминированная по дате) и стрик заходов.
- **Читать** — все 109 писем в ридере: прогресс, закладки, избранное.
- **Темы** — 10 принципов стоицизма с подборками цитат и ссылками на письма.
- **Дневник** — запись практики: ситуация → принцип → вывод.
- **Избранное** — сохранённые цитаты и письма.

## Разработка

```bash
npm install
npm run dev      # http://localhost:5173
npm test         # vitest (логика + целостность данных)
npm run build    # tsc --noEmit && vite build → dist/
```

## Контент

Текст писем парсится из исходного .txt (Windows-1251) разово:

```bash
npm run parse    # или: python3 scripts/parse_letters.py [путь_к_txt]
```

Скрипт режет книгу на письма, конвертирует римские номера, чистит сноски и пишет
`src/data/letters.json`. Цитаты (`quotes.json`) и темы (`themes.json`) курируются
вручную; целостность ссылок между ними проверяется тестом `src/data/data.test.ts`.

## Деплой

Пуш в `main` → GitHub Actions (`.github/workflows/deploy.yml`) собирает и
публикует на GitHub Pages. `base` в `vite.config.ts` — `/personal_seneca/`;
поменяй, если имя репозитория другое.
